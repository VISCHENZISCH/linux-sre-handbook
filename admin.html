Administration Avancée

Optimisation, durcissement et maîtrise des mécanismes bas niveau du système d'exploitation Linux.

~15 min de lecture SRE / Ingénierie Mis à jour Mai 2026

Sécurité & Hardening

Réduire la surface d'attaque par le durcissement système et applicatif.

SSH Hardening

# /etc/ssh/sshd_config
PermitRootLogin no
PasswordAuthentication no
Port 2222
AllowUsers deployer admin

Firewall (UFW)

# ufw default deny incoming
# ufw allow 2222/tcp
# ufw allow 80,443/tcp
# ufw enable

Fail2Ban

# apt install fail2ban
# Protection contre le brute force
# fail2ban-client status sshd

Standards & Outils de Hardening

  • CIS Benchmarks — Normes de durcissement universelles reconnues par l'industrie (Linux, K8s, Cloud).
  • Lynis Audit Tool — Outil open source d'audit de sécurité et de durcissement Linux.
  • Linux Hardening Guide — Guide technique complet sur la protection du noyau.
  • NIST SP 800-123 — Recommandations fédérales américaines pour la sécurité des serveurs généraux.

Packages & Dépendances

Gestion propre des logiciels et des mises à jour de sécurité.

# apt update && apt upgrade -y
# apt-mark hold nginx  # Bloquer la version
# apt autoremove --purge

Systemd Avancé

Maîtriser les unités de service pour une résilience maximale et le sandboxing.

Exemple d'Unit File Robuste (Sandboxed) 
# /etc/systemd/system/webapp.service
[Unit]
Description=Webapp Python
After=network.target postgresql.service

[Service]
User=webapp
Restart=on-failure
RestartSec=5s
CapabilityBoundingSet=CAP_NET_BIND_SERVICE
PrivateTmp=true
ProtectSystem=strict
ProtectHome=yes
NoNewPrivileges=yes

[Install]
WantedBy=multi-user.target

Réseau Avancé

Diagnostic et optimisation de la pile TCP/IP.

# ip addr show
# tcpdump -i eth0 port 80 -n
# dig google.com @8.8.8.8

Stockage & Systèmes de fichiers

Gestion dynamique des volumes avec LVM et santé des disques.

# pvcreate /dev/sdb
# vgcreate data_vg /dev/sdb
# lvcreate -L 50G -n data_lv data_vg
# lvextend -L +10G /dev/data_vg/data_lv --resizefs

Monitoring & Observabilité

Passer de la simple surveillance à la compréhension profonde du système.

Prometheus

Collecte de métriques temporelles via Node Exporter sur les serveurs Linux.

Grafana

Visualisation et dashboards haute performance connectés à Prometheus.

Loki

Agrégation de logs cloud-native et intégration avec la stack PLG.

Ressources de Monitoring & Observabilité

  • Prometheus Docs — L'outil de référence pour l'alerte et la collecte de métriques temporelles.
  • Grafana Docs — Guide de création de dashboards d'observabilité haute performance.
  • OpenTelemetry — Le standard unifié pour les traces, métriques et logs applicatifs.
